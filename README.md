Menu Interativo de Banco – Java

Um sistema simples de menu bancário no terminal, permitindo visualizar saldo, realizar depósitos e saques.

---

Funcionalidades
- Consultar saldo
- Depositar valores
- Sacar valores (com verificação de saldo)
- Menu interativo com repetição até o usuário encerrar

---

Tecnologias utilizadas
- Java
- Scanner
